## Demo
[https://butomania.herokuapp.com/](https://butomania.herokuapp.com/)

## Technologies
* Front-end
  * React
  * React Router Dom
  * React Hooks
  * React Redux && Redux Thunk
  * React Loader Spinner
  * React Toastify
  * Axios
  * Prop Types
  * Scss

* Back-end
  * Express.js
  * Mongoose
  * JsonWebToken
  * Bcrypt.js
  
## Features

* Fetching products from database
* Adding/Removing product from basket
* Changing products quantity in basket
* Simple PageHero slider
* Viewing full details of products
* Responsive Design
* Login/Register/Address validation
* Filtering products by name/brand/categories/type/price/color/size
* Private Route
* User Authentication
* Hashing user password
* Changing user data and delivery address
